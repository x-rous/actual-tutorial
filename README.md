# Actual Budget Tutorial

This repository contains a single-page, bilingual tutorial for Actual Budget. It walks through the app’s core workflows, including:

- Tracking Budget vs Envelope Budget
- Setting up accounts and categories
- Credit cards, transactions, rules, and reports
- Keyboard shortcuts and common workflow questions

## Tutorial

Open the live tutorial here:

[https://x-rous.github.io/actual-tutorial/](https://x-rous.github.io/actual-tutorial/)

## Notes

- The tutorial is designed as a guided, end-user walkthrough.
- The page is built as a static `index.html` file.
